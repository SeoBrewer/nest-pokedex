# Run in Development

1. Clone the repository

```
2. Execute

```

yarn install

```

3. Nest CLI must be installed globally

```

npm i -g @nestjs/cli

```

4. Run the DB

```

docker-compose up -d

```

## Used Stack

- MongoDB
- NestJS
```
